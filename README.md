## Description
There are 2 folders only-rdbms and multi-rdbms, both have the same function except in the way of storing and organizing data
# Running
## Only-rdbms
```bash
cd only-rdbms
npm install
npm run start
```
## Multi-dbms
```bash
cd multi-rdbms
npm install
npm run start
```
